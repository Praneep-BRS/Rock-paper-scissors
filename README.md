# Rock Paper Scissors Game

### Introduction

This project is a simple implementation of the classic Rock Paper Scissors game designed to be played against a computer opponent. The game is built using HTML, CSS, and JavaScript. Players can enter their names, make choices, and compete against the computer to determine the winner.

### Features

- Player input: Enter your name before starting the game.
- Game initiation: Click "Start Game" to begin playing.
- Player choices: Select Rock, Paper, or Scissors.
- Score tracking: Keep track of player and computer scores.
- Winner determination: Decide the winner based on choices made.
- Restart option: Restart the game to play again.
- Theme selection: Choose from different color themes.

---Tech Stack:---
HTML, CSS, Javascript
